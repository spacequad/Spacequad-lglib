lgLib Configuration

The lgLib plugin provides certain functions, javascript, and CSS that is commonly used across plugins.
General Settings
Calendar Style Select the visiual style to be used for the popup calendar.

After changing this option, you must clear the layout cache
Path to display images Enter the path for the plugin to create images for display. This path is relative to the plugin's HTML directory, so the resulting image url is "http://yoursite.com/lglib/path/you/enter"
Scheduled Task Interval This plugin includes a specialized cron.php file that can be called locally or remotely to trigger scheduled tasks. Use this if you're concerned about a) irregular execution via normal site visits, or b) have lengthy tasks to perform and don't want to penalize your site visitors.

To use this feature, set this value to some number of seconds (e.g. 3600 for hourly jobs) and set the global site cron interval to zero. Then, via a host cron job or remote monitoring tool, execute /lglib/cron.php. For example, you could set your remote monitoring tool to call http://yoursite/lglib/cron.php every 5 minutes. All scheduled tasks will be executed from this tool rather than by your site visitors.

If you leave the global site cron interval set, regular scheduled tasks will be spawned by site visitors as usual. Database backups are always handled by lglib's cron handler due to the time required.
Scheduled Task Security Key As a basic security measure, you can protect the scheduled task URL by providing some random string. This string must be included with the URL (as "http://yoursite/lglib/cron.php?key=XXXXX") or as an argument to the command line ("php lglib/cron.php XXXXX"). If this value is configured but not supplied when cron.php is invoked, no action will be taken.
Image Cache Cleaning Interval Enter a number of minutes to wait between cache cleaning cycles. To minimize load and wait time for the visitor, the image cache is only cleaned every X minutes and only one cache directory out of 16 is cleaned at a time. Enter zero to disable cache cleaning.
Cached Image Max Age Enter the number of days that a cached image may remain before being automatically removed. A small number here will keep disk usage low, but may increase wait times for visitors as images are recreated. A value of zero disables cache cleaning.
